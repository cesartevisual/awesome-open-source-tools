## 3D & motion

- [Blender](https://github.com/blender/blender) — 3D modeling, animation, and rendering for product viz, motion, and asset work.

## Video & streaming

- [HandBrake](https://github.com/HandBrake/HandBrake) — Video transcoder for compressing demos and deliverables.
- [Kdenlive](https://github.com/KDE/kdenlive) — Non-linear video editor for demos, tutorials, and marketing cuts.
- [OBS Studio](https://github.com/obsproject/obs-studio) — Screen capture and live streaming for tutorials and design reviews.

## Design & UI engineering

- [Responsively App](https://github.com/responsively-org/responsively-app) — Multi-device responsive preview browser (mirrored interactions, shared inspector, device profiles).
- [ScreenForge](https://github.com/nclandrei/screenforge) — CLI for App Store–style marketing screenshots (device frames, backgrounds, overlays).

## Screencasts & demos

- [Gifski](https://github.com/sindresorhus/Gifski) — High-quality GIF encoder for Mac (wraps the gifski library).
- [Hide Icons](https://github.com/com-entonos/Hide-Icons) — Menu bar toggle to hide desktop icons for clean screenshots.
- [KeyCastr](https://github.com/keycastr/keycastr) — On-screen keystroke overlay for screencasts and presentations.

## Color

- [Pika](https://github.com/superhighfives/pika) — Native color picker with multiple formats and screen sampling.

## Office & docs

- [LibreOffice](https://github.com/LibreOffice/core) — Office suite (docs, slides, sheets) for specs, decks, and stakeholder deliverables.

## Browsers

- [Zen Browser](https://github.com/zen-browser/desktop) — Firefox-based browser with workspaces, split view, and compact mode for focused design research.

## Development

- [CodexBar](https://github.com/steipete/CodexBar) — Menu bar usage meters and reset timers for AI coding providers (Codex, Claude, Cursor, etc.).
- [Warp](https://github.com/warpdotdev/Warp) — Agentic terminal for design engineering workflows (client is OSS; AGPL/MIT split).

## macOS — Menu bar & system

- [Clocker](https://github.com/n0shake/clocker) — World clock and time-zone planner for distributed teams.
- [Hidden Bar](https://github.com/dwarvesf/hidden) — Collapse low-priority menu bar icons behind a chevron.
- [MeetingBar](https://github.com/leits/MeetingBar) — Calendar meetings in the menu bar with one-click join (Zoom, Meet, Teams, etc.).
- [MonitorControl](https://github.com/MonitorControl/MonitorControl) — External display brightness, contrast, and volume from the menu bar or keyboard.

## macOS — Windows & workspace

- [Later](https://github.com/LazaroZero1176/later) — Save and restore your Mac app workspace from the menu bar.
- [Rectangle](https://github.com/rxhanson/Rectangle) — Window snapping and keyboard shortcuts (Spectacle successor).

## macOS — Files, input & emulation

- [Enjoyable](https://github.com/pedroh77/enjoyable-silicon) — Map game controllers to keyboard and mouse (Apple Silicon fork of the classic app).
- [OpenEmu](https://github.com/OpenEmu/OpenEmu) — Emulator front-end for reference and playtesting.
- [unar](https://github.com/MacPaw/XADMaster) — Open-source extraction engine behind The Unarchiver (`unar` CLI).

## Free (not open source)

### Audio

- [Elgato Wave Link](https://www.elgato.com/us/en/s/wave-link-app) — Creator audio mixer (routing, VST/AU effects). Free; proprietary.

### Media

- [Plex](https://www.plex.tv/) — Media server and clients. Freemium; proprietary server.

### System & utilities

- [coconutBattery](https://www.coconut-flavour.com/coconutbattery/) — Battery health stats for Mac and iOS devices. Free; proprietary.
- [The Unarchiver](https://theunarchiver.com/) — Archive extractor app. Free; proprietary — engine is [unar](https://github.com/MacPaw/XADMaster).
